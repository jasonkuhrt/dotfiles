# Library ADT Plugin

**Status: DISABLED** - Enable in plugin.json when needed.

Plugin for creating and working with ADT (Algebraic Data Type) patterns - discriminated unions with Effect Schema or similar libraries.

## Skills

- **creating-adts** - Scaffold ADT unions with proper member files and barrel exports

## Convention Reference

See `~/.claude/docs/conventions/` for documentation:
- `namespace-module.md` - Core `_.ts`/`__.ts` pattern
- `library-adt.md` - ADT-specific patterns

## Enabling

To enable this plugin, edit `.claude-plugin/plugin.json` and set `"enabled": true`.
