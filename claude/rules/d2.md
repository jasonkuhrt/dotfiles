# Diagrams

## D2 is the Source of Truth

**Never manually write or edit ASCII/text diagrams.** D2 is the authoring format. All other formats are rendered outputs.

## Workflow

1. Write/edit the `.d2` source file
2. Render outputs via CLI: `d2 input.d2 output.png` (or `.svg`, `.txt`)
3. Never hand-craft ASCII art â€” if a text version is needed, render it from D2

## Why

- Manual ASCII wastes tokens and drifts from the D2 source
- D2 is the single source of truth; outputs are disposable artifacts
- Edits go in `.d2`, then re-render

## File Naming

```
flow.<name>.d2      # source
flow.<name>.png     # rendered
flow.<name>.txt     # rendered (if needed)
```
