# Skills

* Read the `cc_writing-skills` skill first before any work - it defines naming conventions, structure requirements, and common mistakes

## Role Suffixes

Use these suffixes when a skill's purpose clearly fits one of these roles:

* **teacher** — Educational, documentation, explaining concepts, reference material
* **doctor** — Setup, config checking, onboarding, linting, debugging, diagnostics
