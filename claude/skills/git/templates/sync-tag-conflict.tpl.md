# Tag Conflict

```
Tag conflict: @:TAG_NAME

Local:
@:LOCAL_SHA | @:LOCAL_DATE | @:LOCAL_MESSAGE

Remote:
@:REMOTE_SHA | @:REMOTE_DATE | @:REMOTE_MESSAGE
```

## Example

```
Tag conflict: v1.2.0

Local:
abc1234 | 2025-01-15 | Release v1.2.0 with hotfix

Remote:
def5678 | 2025-01-14 | Release v1.2.0
```
