# Trunk Dirty

```
✗ Cannot sync: @:TRUNK has unpushed commits
@:UNPUSHED_COUNT commit(s) ahead of origin/@:TRUNK:
@...- @:SHORT_SHA | @:DATE | @:MESSAGE
Push or reset before running git-sync.
```

## Example

```
✗ Cannot sync: develop has unpushed commits
2 commit(s) ahead of origin/develop:
- abc1234 | 2025-01-17 | WIP: experimenting
- def5678 | 2025-01-18 | More experiments
Push or reset before running git-sync.
```
