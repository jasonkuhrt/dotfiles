# Cmd UX Future Discussion

This file holds deferred feature directions that are worth revisiting after the current command-engine push.

## Deferred Features

### Offline Synthesis Engine

A periodic offline reviewer should read semantic logs and produce structured proposals for aliases, flows, namespaces, missing capabilities, and workflow compression opportunities.

### Semantic Action Telemetry

The current deterministic engine logs command-centric semantic events well enough for ranking and flow proposals, but the broader intelligence loop still needs richer semantic action telemetry across keymaps, pointer fallbacks, churn signals, and multi-surface workflows.
