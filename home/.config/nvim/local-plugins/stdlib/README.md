# stdlib

Repo-owned Lua standard library for local Neovim plugins.

## Namespace

- `stdlib.*` for host-agnostic helpers
- `stdlib.nvim.*` for Neovim-specific helpers

## Scope

This package is for reusable infrastructure only:

- strings
- collections
- fs/cache helpers
- generic Neovim wrappers

It is not for plugin/domain semantics.
