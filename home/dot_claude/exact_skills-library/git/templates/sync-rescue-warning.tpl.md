# Rescue Warning

```
⚠ Rescue branch created: rescue/@:BRANCH_NAME
@:UNPUSHED_COUNT unpushed commit(s):
@...- @:SHORT_SHA | @:DATE | @:MESSAGE
Cherry-pick: git cherry-pick @...@:SHORT_SHA
```

## Example

```
⚠ Rescue branch created: rescue/feat-HEA-1234
3 unpushed commit(s):
- abc1234 | 2025-01-15 | Add new feature
- def5678 | 2025-01-16 | Fix bug in feature
- ghi9012 | 2025-01-17 | Update tests
Cherry-pick: git cherry-pick abc1234 def5678 ghi9012
```
